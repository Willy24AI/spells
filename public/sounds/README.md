# Sound Effects

This directory should contain the following sound effects:
- correct.mp3: Played when a valid word is found
- incorrect.mp3: Played when an invalid word is submitted
- pangram.mp3: Played when a pangram is found
- game-over.mp3: Played when the game ends

Note: Sound files are not included in the repository. You should:
1. Use royalty-free sound effects
2. Keep files small (< 100KB each)
3. Use MP3 format for best browser compatibility
